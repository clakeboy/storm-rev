生成 git commit message 时：
- 使用**全中文**
- **必须准确描述本次实际修改的功能**，禁止使用推测或不相关的内容
- 类型前缀使用中文：`新增`、`修复`、`优化`、`重构`、`移除`
- 格式：`类型: 简短描述`

**示例**：
- ✅ `新增: 封面生成配置功能`
- ✅ `修复: 登录积分未到账`
- ✅ `优化: AI 测试页模型列表`
- ✅ `重构: 生图配置集中管理`
- ❌ `feat: 新增邮箱配置 API`（禁止英文类型）
- ❌ `修复: 修复问题`（禁止空泛描述）
